Introduction

This is the documentation about the project Arcade It's a project that we need to build a Arcade borne, that allowed us to switch game and graphics

Donwload requirements for launch our program

You need to have SFML Library and Ncurses in the laptop

Download SDL2

sudo dnf install SDL2-devel

Download SDL2 Image

sudo dnf install SDL2_image-devel

Download SDL2 FONT TTF

sudo dnf install SDL2_ttf-devel

Launch program

./arcade {graphicLibrayName}
